
# Daily Quotes Telegram Bot

Простой Telegram-бот, который отправляет цитаты пользователям.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env` и добавьте в него ваш токен бота:

```
BOT_TOKEN=ваш_токен
```

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — запускает бота
